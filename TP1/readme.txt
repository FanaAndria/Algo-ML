Ce code implémente un jeu de puzzle glissant (comme le "8-puzzle") où
le joueur peut choisir une grille 3x3 ou 4x4,
 déplacer les tuiles pour résoudre le puzzle,
 ou utiliser une option automatique pour le résoudre via l'algorithme A*.
